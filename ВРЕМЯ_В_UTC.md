# Время в карточках теперь в UTC ✅

**Дата:** 18 января 2026  
**Статус:** Реализовано

## Изменение

Карточки predictions теперь показывают время в **UTC** вместо локального времени.

## Причина

UTC - универсальное координированное время, одинаковое для всех пользователей по всему миру. Это важно для:
- Синхронизации наблюдений между пользователями
- Корреляции с событиями (землетрясения, крипто движения)
- Профессионального подхода к мониторингу

## Реализация

**Файл:** `web/static/index.html`  
**Функция:** `formatDetectionTime(timestamp)`

### Было (локальное время):
```javascript
const timeStr = date.toLocaleTimeString('en-US', { 
    hour: '2-digit', 
    minute: '2-digit',
    hour12: false 
});
```

### Стало (UTC):
```javascript
const hours = String(date.getUTCHours()).padStart(2, '0');
const minutes = String(date.getUTCMinutes()).padStart(2, '0');
const timeStr = `${hours}:${minutes} UTC`;
```

## Формат отображения

- **Сегодня:** `Today · 19:45 UTC`
- **Вчера:** `Yesterday · 14:30 UTC`
- **Другие дни:** `Jan 17 · 08:15 UTC`

## Проверка

Откройте PWA: http://localhost:5555

Все карточки predictions теперь показывают время с меткой `UTC`.

## Примечание

Anomaly Clusters продолжают показывать локальное время, так как это события в реальном времени для текущего пользователя. Predictions показывают UTC, так как это исторические паттерны для глобального анализа.
